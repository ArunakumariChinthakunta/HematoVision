# HematoVision Blood Cell Classification
HematoVision is a web-based application developed to simplify blood cell data management and reporting.
It allows users to upload and store blood sample details securely with role-based access.
The system provides an easy interface for classification, report generation, and viewing results.
This project enhances laboratory efficiency by reducing manual work and improving record accuracy.
It ensures secure, fast, and organized handling of blood cell information for medical use.
